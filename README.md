# Kaleido TypeScript SDKs

This repository contains Kaleido's SDKs for TypeScript.

## Available SDKs

### Workflow engine SDK

The workflow engine SDK is available under `packages/workflow-engine-sdk`.

To get started with an example project using the workflow engine SDK, you can run:
```
npx @kaleido-io/workflow-engine-sdk init <your-project-name>
```

Alternatively you can to install the package with:

```
npm install @kaleido-io/workflow-engine-sdk
```
